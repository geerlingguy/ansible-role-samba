# Ansible Role: Samba (SMB)

[![Build Status](https://travis-ci.com/geerlingguy/ansible-role-samba.svg?branch=master)](https://travis-ci.com/geerlingguy/ansible-role-samba)

Installs Samba client and server for RHEL/CentOS.

## Requirements

Samba requires ports 137, 138, 139, 445 to be open to function properly. For easy firewall configuration, you can use the `geerlingguy.firewall` role.

## Role Variables

None.

## Dependencies

None.

## Example Playbook

    - hosts: servers
      roles:
        - geerlingguy.samba
